## Descripción
Este proyecto implementa un **sistema de subastas para objetos de colección** desarrollado en **Java** como una aplicación de consola.
El sistema permite:
- Registrar usuarios
- Autenticar usuarios
- Crear subastas
- Listar subastas activas
- Realizar ofertas sobre subastas
El desarrollo aplica principios de **Programación Orientada a Objetos**, incluyendo:
- Encapsulación
- Herencia
- Separación de responsabilidades
- Uso de colecciones dinámicas (`ArrayList`)
---
## Arquitectura del sistema
El sistema sigue una **arquitectura por capas**:
- cr.ac.ucenfotec.subastas
  - model: Clases del dominio del sistema
  - service: Lógica del negocio
  - ui: Interfaz de usuario en consola
---
## Ejecución del programa
El programa se ejecuta desde la clase: Main
La aplicación corre en consola y permite navegar por el sistema mediante un **menú interactivo**.
